📦 主要脚本
  • test_hands.sh       - 完整运动测试
  • reset_hands.sh      - 重置到初始状态
  • restart_system.sh   - 重启系统

🚀 快速开始
  # 完整测试 (双手,所有手指,2秒延迟)
  ./test_hands.sh
  
  # 快速测试 (1秒延迟)
  ./test_hands.sh --delay 1.0
  
  # 重置手指
  ./reset_hands.sh

🎮 常用命令
  # 只测试右手
  ./test_hands.sh --hand right
  
  # 只测试食指
  ./test_hands.sh --fingers index
  
  # 只测试大拇指
  ./test_hands.sh --fingers thumb
  
  # 测试多个手指
  ./test_hands.sh --fingers index,middle,thumb

📊 测试内容
  四指 (index/middle/ring/little):
    ✓ 3步弯曲: 180° → 150° → 120° → 87°
    ✓ 3步伸直: 87° → 120° → 150° → 180°
  
  大拇指侧摆 (thumb_1):
    ✓ 3步外展: 0 → 0.4 → 0.8 → 1.16 rad
    ✓ 3步复原: 1.16 → 0.8 → 0.4 → 0 rad
  
  大拇指弯曲 (thumb_2):
    ✓ 3步弯曲: 101° → 115° → 127° → 140°
    ✓ 3步伸直: 140° → 127° → 115° → 101°

⏱️ 预计时间
  双手全部手指 (delay=2.0): ~2分钟
  单手测试 (delay=2.0):     ~1分钟
  快速测试 (delay=1.0):     ~1分钟
  大拇指测试 (delay=2.0):   ~0.4分钟
